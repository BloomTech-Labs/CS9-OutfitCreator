# /local-auth
## /local-auth/signup
POST - Register a User
This request should have the following format:
```
{
    "username": "ellen",
    "password": "password123",
    "email": "ellen@email.com"
}
```
## /local-auth/login
POST - Login a User
This request should have the following format:
```
{
    "username": "ellen",
    "password": "password123",
    "email": "ellen@email.com"
}
```
